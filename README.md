
<h2 align="center">Simple shell program written in C.</h2>

## Features
* Run Simple Commands 
    - cd
    - exit

* Global Variables
    - $my_var="Hello World!"
    - $echo $my_var

* Signal Handling
    - 1. CTRL-S: freeze screen
    - 2. CTRL-Q: unfreeze screen
    - 3. CTRL- Z: push process to background
    - 4. fg: bring process from  backround to foreground
