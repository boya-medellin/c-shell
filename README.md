
#Simple shell written in C.

## Features
* Run Simple Commands 
    - cd
    - exit

* Global Variables
    - $my_var="Hello World!"
    - $echo $my_var

* Signal Handling
    - CTRL-S: freeze screen
    - CTRL-Q: unfreeze screen
    - CTRL- Z: push process to background
    - fg: bring process from  backround to foreground
